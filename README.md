
# Sealion

Sealion is a 2D game engine written in C.

## Builing from Source

### Build requirements
* Python3 (Any version will do)
* GCC (Or any C compiler of your choice)
* Cygwin (Currently only build method supported)

### Building The DLL libarary
<code>make lib</code> 

### Build The demo
<code>make game</code>
